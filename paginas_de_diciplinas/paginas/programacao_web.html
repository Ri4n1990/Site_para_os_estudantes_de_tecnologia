Programação Web

o que você vai aprender?

Na diciplina de programação web, você aprenderá a criar aplicativos e sites dinâmicos e interativos. Ao longo do curso, você será introduzido aos conceitos fundamentais da linguagem PHP e sua integração com HTML, CSS e bancos de dados. Você aprenderá a desenvolver sistemas de login, registro de usuários, envio de formulários, manipulação de dados e criação de páginas dinâmicas.
Além disso, será ensinado sobre a segurança na programação web, como proteger os dados do usuário e evitar vulnerabilidades. Você também terá a oportunidade de aprender práticas recomendadas para desenvolver aplicativos web escaláveis e de alta qualidade, levando em consideração boas práticas de codificação e organização de projetos.

passos para aprender

  • Introdução ao PHP: conceitos básicos da linguagem, sintaxe e estrutura de um arquivo PHP.
  • Integração com HTML e CSS: como incorporar código PHP em arquivos HTML e estilizar páginas usando CSS.
  • Manipulação de formulários: processamento de dados enviados por formulários, validação e exibição de erros.
  • Banco de dados e SQL: interação com bancos de dados, criação de consultas SQL para recuperar, inserir, atualizar e excluir dados.
  • Sistemas de autenticação e autorização: desenvolvimento de sistemas de login e controle de acesso baseados em PHP.
  • Tratamento de erros e exceções: identificação e tratamento de erros para garantir a estabilidade e segurança do aplicativo.
  • Segurança na programação web: práticas recomendadas para proteger dados sensíveis, evitar ataques e garantir a integridade do sistema.
  • Boas práticas de desenvolvimento: organização de código, reutilização de funções, modularidade, padronização e documentação do código.
